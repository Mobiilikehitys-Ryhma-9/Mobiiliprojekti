# Projektin kuvaus

Olemme toteuttaneet esteettömään liikkumiseen kaupunkiympäristössä edistävän sovelluksen React Nativella. Karttanäkymässä esitetään reittivaihtoehtoja kävelevälle tai pyörätuolilla liikkuvalle käyttäjälle. Rekisteröidyt käyttäjät pystyvät ilmoittamaan omia esteettömyyteen liittyviä huomioita, jotka huomioidaan reittivaihtoehdoissa. Esteet voivat olla väliaikaisia, tai joiden kesto voi olla vaikea ennalta määrittää.

Reittivaihtoehtojen hakemiseen sovellus käyttää [OpenRouteService](https://openrouteservice.org/) rajapintaa.
Käyttäjien tehdyt ilmoitukset tallentuvat Googlen Firebase Firestoreen.
